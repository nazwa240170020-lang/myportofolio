Portfolio Website

Portfolio Website ini dirancang sebagai website pribadi untuk menampilkan profil, project, sertifikat, serta aktivitas akademik. Website ini dibuat dengan tampilan modern, animasi interaktif, dan desain responsif sehingga dapat diakses dengan baik di berbagai perangkat. Website ini juga menampilkan beberapa fitur interaktif seperti theme toggle, animasi partikel, efek scroll reveal, serta tampilan project showcase untuk memperlihatkan hasil karya secara profesional.
